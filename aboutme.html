About me

I have experienced Art and Graphics during my education in secondary school and in an advanced course in which I have took part in the 2012.
Given my passion for Graphics Design, I decided to start learning how to program with HTML and CSS. Once I was confident using these languages I decided to learn some back-end programming with PHP and using MySQL as the database. I am now focusing on using a combination of these languages to develop some projects.
I believe that programming opens your mind in way that not a lot of activities can.
I enjoy the time I spend programming and learning new methods to write code even if sometimes it’s not as easy and doesn’t go as planned.
